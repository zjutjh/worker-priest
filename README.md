# api-recreate
>api重构
>

本项目是`微精弘`将来要依赖的一个`应用接口`

采用`mvc架构`，而`视图`部分将不会出现在架构中（面向其他项目的视图开放固定的几个`接口`）

将会实现以下几个`功能`：

* 饭卡记录查询
* 图书馆书籍检索/借阅记录查询
* 正方/原创系统 成绩明细/班级信息/考试信息查询
